✨ 1. Text Formatting

Convert your normal text into Unicode formatted text that works on LinkedIn, Twitter, Facebook, and more.

Bold — B button or Ctrl + B

Select text → Click B or press Ctrl + B

Input: Hello World

Output: 𝐇𝐞𝐥𝐥𝐨 𝐖𝐨𝐫𝐥𝐝

Italic — I button or Ctrl + I

Select text → Click I or press Ctrl + I

Input: Hello World

Output: 𝐻𝑒𝑙𝑙𝑜 𝑊𝑜𝑟𝑙𝑑

Bold Italic — BI button or Ctrl + Shift + B

Select text → Click BI

Input: Hello World

Output: 𝑯𝒆𝒍𝒍𝒐 𝑾𝒐𝒓𝒍𝒅

Underline — U button or Ctrl + U

Select text → Click U

Input: Hello World

Output: H̲e̲l̲l̲o̲ W̲o̲r̲l̲d̲

Strikethrough — S button or Ctrl + Shift + X

Select text → Click S

Input: Hello World

Output: H̶e̶l̶l̶o̶ W̶o̶r̶l̶d̶

💡 Tip: Click the same format button again to remove formatting (toggle).

😊 3. Emoji Picker

How to use

Click the 😊 button → Search or browse emojis → Click any emoji to insert at cursor position

  • Search emoji by name (type "fire" to find 🔥)
  • Browse 1000+ emojis organized by category
  • Click to insert at cursor position
  • Click outside or press Esc to close

🔤 4. Case Converter

UPPERCASE — AB or Ctrl + Shift + U

Converts selected text (or all text) to UPPER CASE

Input: hello world → Output: HELLO WORLD

lowercase — ab or Ctrl + Shift + L

Converts selected text to lower case

Input: HELLO WORLD → Output: hello world

Title Case — Ab or Ctrl + Shift + T

Converts selected text to Title Case

Input: hello world → Output: Hello World

🔍 8. Find & Replace

How to use

Click the 🔍 button or press Ctrl + H

  • Find: Type the text you want to find
  • Replace: Type the replacement text
  • Find Next: Jump to next occurrence
  • Replace: Replace current occurrence
  • Replace All: Replace all occurrences at once

Press Enter in the find field to jump to next match.

Ω 9. Special Characters

How to use

Click the Ω button → Browse categories → Click any character to insert

Categories:

  • Arrows (← → ↑ ↓)
  • Math (± × ÷ ≠ ≈)
  • Currency ($ € £ ¥ ₹)
  • Symbols (© ® ™ §)
  • Punctuation (— – · « »)
  • Fractions (½ ⅓ ¼)
  • Shapes (● ○ ◆ ■ ▲ ★)
  • Checks (✓ ✔ ✗ ✅ ❌)
  • Stars (★ ☆ ✦ ✧ ✨)

🧹 10. Text Cleaner

How to use

Click the 🧹 button to automatically clean your text:

  • Removes trailing spaces from each line
  • Collapses 3+ blank lines into 2
  • Removes zero-width invisible characters

  • Adds space before emoji if missing
  • Adds space after emoji if followed by text

↩️ 12. Auto Line-Break Fix

How to use

Click the ↩️ (second one) to normalize line breaks:

  • Normalizes all line endings to standard format
  • Removes zero-width characters
  • Removes trailing whitespace per line
  • Collapses multiple blank lines
  • Removes leading/trailing blank lines

✍️ 13. Custom Signature (Pro)

How to use

Click the ✍️ button → Fill in your details → Save & Insert

  • Name: Your name or brand (e.g., "Link Polish")
  • Tagline: Your title or tagline (e.g., "LinkedIn Text Formatter")
  • CTA: Call to action (e.g., "Try it free → linkpolish.com")

Buttons:

  • 💾 Save: Save signature for future use
  • Insert at End: Add signature to end of current post
  • Auto-insert: Automatically add signature to every post when copying
💡 Tip: Enable "Auto-insert" so your signature is always included when you copy formatted text.

📥 14. Export as .txt

How to use

Click the 📥 button to download your post as a .txt file

  • File is named with today's date: linkpolish-post-2026-04-26.txt
  • Useful for saving drafts or backing up posts

📋 15. Copy Formatted Text (Pro)

How to use

Click the big blue "Copy Formatted Text" button

  • Copies your formatted post to clipboard
  • Paste directly into LinkedIn, Twitter, etc.
  • Includes auto-signature if enabled
  • Button turns green and says "Copied!" on success
⚠️ Note: Copy Formatted Text is a Pro feature. Purchase a license or use the unlock code to activate.

👁 16. Live Preview

How it works

The right panel shows a live LinkedIn preview of your post:

  • Updates in real-time as you type
  • Shows how your post will look on LinkedIn
  • Includes mock profile, like/comment counts
  • On mobile: switch between Editor and Preview tabs

📊 17. Live Stats Bar

What it shows

  • Chars: Total character count
  • Words: Total word count
  • Lines: Total line count
  • Read time: Estimated reading time (based on 3.5 words/sec)
💡 Tip: LinkedIn posts with 1,300-2,000 characters tend to get the most engagement. Keep read time under 45 seconds for best results.
